# eis-app

## Pre-requisites

You have to install:
- [Maven](https://maven.apache.org/) 
- [JDK 8](https://www.oracle.com/java/technologies/downloads/)

## Usage

Build the project with

```bash
mvn package 
```

Run the jar file with for help

```bash 
java -jar target/eis-app-${project.version}.jar --help
```
